# Express + SQL Training

A simple training project demonstrating how to connect and interact with a MySQL database using Express.js.

## Tech Stack
- **Express.js**: Web framework
- **MySQL2**: Database client
- **Dotenv**: Environment configuration

## Getting Started

1. **Install dependencies**
   ```bash
   npm install
   ```

2. **Configure Environment**
   Ensure you have a `.env` file with your database credentials.

3. **Run the project**
   ```bash
   npm run start:dev
   ```

## Description
This repository serves as a practice ground for handling SQL operations within an Express.js application.
